# CH32X035

SWDIO - PC18/DIO @ MCU
SWCLK - PC19/DCK @ MCU

PC17 to High, enable USB ISP DOWNLOAD MODE

## Boards

Official CH32X035EVT

- CH32X035F8U6-R0
- CH32X035G8U6-R0
- CH32X035C8T6-R0

LED1, LED2 blue LED, active low, open, connected to open jumper
